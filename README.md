## Nano测序原理
#### 测序原理
一张芯片上有2048个membrane wells(孔)，每个孔上有个reader

每四个wells共享一个Amplifier（信号放大器），即512个放大器，512组wells

在启动测序仪后，机器自检，会将每组 wells 中依据效率高低排序。

测序开始，仪器先用每组 wells 中效率最高的 wells，运行 8 小时后，更换效率第二的，以此类推。
#### 建库方法
1. 1D文库
   - 定义：正反链，分别测序
   - 碱基判读准确率：85% 
   - 流程：
     - 打断
     - 末端+A
     - 连接Adapter（带有Motor）
     - 接头中有段序列与Tether结合，吸附DNA到膜上
2. 转座酶建库
   - 定义：使用带有Adapter的转座酶打断DNA
   - 流程：
     - 转座酶打断
     - 接头上加入Moter
3. 1 $D_2$ 文库
   - 定义：使用$D_2$接头，让第二链紧跟第一链测序
   - 碱基判读准确率：90% 
   - 缺点：由于文库质量，蛋白活性等因素，导致并不是所有的第一链后都会测到第二链
#### 造成 wells 失效的原因：
1. 无电信号:
   - wells 中没有 Reader 蛋白
   - 纳米孔不通
2. 强电信号:
   - 膜破损
3. 电信号互相干扰
   - 在单个well中有两个及以上的 Reader 蛋白
#### 碱基判读
1. 原理
   - R9每次判读5个碱基信号（根据跨膜长度，共1024种情况）
2. 误读情况
   - 空间结构相似性，嘌呤间误读，嘧啶间误读更容易发生
   - 碱基复杂度低的序列（如，polyA序列），更容易误读
#### 工具
1. Reader
   - 跨膜蛋白
2. Membrane
   - 人工合成的多聚物膜
   - 高电阻率
3. Motor
   - 接头上的动力蛋白
   - 解开双螺旋
   - 提供推动力
4. Tether
   - 锚定DNA或RNA链,使其进入纳米孔中
#### 仪器类型
1. PromehION
2. GridION
3. Type C
4. MnION
